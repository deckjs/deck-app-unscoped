# deck-app-unscoped 

This allows a Node 0.10 (or npm 1) users to install `@deck/app`

[![js-standard-style](https://img.shields.io/badge/code%20style-standard-brightgreen.svg?style=flat)](http://standardjs.com/)


## Install

```sh
npm i -g deck-app # node .10
```
